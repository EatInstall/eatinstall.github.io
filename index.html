Eat - The Open-Source Package Manager for Linux

Eat is an open-source Package Manager for Linux, a clone of Unix.

Eat provides many ways to install programs without modifiying your PATH. Instead, it just creates Bash aliases. You can modify your shell's init script to add the program.

To install Eat, just run this command in your Linux command-line:

bash -c "$(curl -fsSL https://raw.githubusercontent.com/Tyler887/eat/main/inst-script.sh -#)"
